📘 Öğrenci Not Sistemi

Bu proje, öğrenci notlarını tutmak ve yönetmek için geliştirilmiş basit bir ASP .NET Core MVC (6.0) öğrenci not kayıt sistemi uygulamasıdır.

📝 Proje Hakkında

Öğrenci Not Sistemi, öğrencilerin not kayıtlarını oluşturmanızı, düzenlemenizi ve görüntülemenizi sağlayan bir web uygulamasıdır. Proje Visual Studio kullanılarak geliştirilmiş olup C# dili ile yazılmıştır.

🚀 Özellikler

Öğrenci notlarını kaydetme

Öğrenci bilgilerini listeleme

Yeni öğrenci ekleme

Var olan öğrenci bilgilerini düzenleme

Öğrenci ortalamalarını hesaplama

🧰 Kullanılan Teknolojiler

✔️ C#

✔️ ASP .NET Core MVC (6.0)

✔️ HTML / CSS
